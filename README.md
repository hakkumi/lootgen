# LootGen

## Описание
Консольная утилита для симуляции вероятности выпадения редких игровых предметов (лута) из сундуков. Используется для баланса игры.

## Технологии
- Python 3.12
- rich (красивый вывод таблиц)
- ruff (линтер)
- GitHub Actions (CI/CD)
